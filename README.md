## 專題名稱
Game Night

## 使用工具
- **Qt Creator**
- **C++ program language**

## 功能說明
1. 首頁功能：
   - 使用者可輸入名稱。
   - 提供選擇兩個小遊戲的介面。
   - 查看最高分數。

2. 小遊戲功能：
   - 點擊按鈕可進入個別小遊戲。
   - 各小遊戲內具備多樣化的遊戲元素。
   - 支援透過鍵盤操作進行互動。
